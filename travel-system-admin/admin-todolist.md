# 管理端 TodoList（基于 Ruoyi）

## 1. 管理端基础配置

- [ ] **菜单与权限体系规划**
  - [ ] 设计后台菜单结构：用户管理、内容管理、线路管理、统计分析、系统设置等
  - [ ] 为不同角色分配权限：超级管理员、内容运营、数据分析等
  - [ ] 与后端接口权限注解保持一致

- [ ] **主题与品牌化**
  - [ ] 配置系统标题与 Logo
  - [ ] 配置登录页文案与背景图（可选）
  - [ ] 设置中英文显示（可选）

## 2. 用户管理功能

- [ ] **用户列表**
  - [ ] 展示用户基础信息（昵称、头像、城市、注册时间、状态）
  - [ ] 支持条件筛选（昵称、城市、状态、时间区间）
  - [ ] 支持导出用户列表为 Excel

- [ ] **用户详情与控制**
  - [ ] 查看用户行为概要（游记数、评论数、打卡数）
  - [ ] 启用/禁用用户账号
  - [ ] 标记高价值用户 / 风险用户（可扩展字段）

- [ ] **违规处理支持**
  - [ ] 与游记/评论举报信息联动，查看用户违规情况
  - [ ] 根据违规程度手动禁言/封禁（视需求）

## 3. 内容管理（游记、评论、举报）

- [ ] **游记审核管理**
  - [ ] 待审核游记列表（标题、作者、城市、提交时间）
  - [ ] 游记详情预览（图文内容、图片、关联景点）
  - [ ] 审核操作：通过 / 拒绝（附原因）
  - [ ] 批量审核功能（可选）

- [ ] **游记内容运营**
  - [ ] 将优质游记设为“精选”，在前端首页或专题展示
  - [ ] 可编辑游记标题、摘要（不改用户原文，可加运营注释）
  - [ ] 删除严重违规游记

- [ ] **评论与举报管理（可扩展）**
  - [ ] 评论列表查看与删除
  - [ ] 举报记录管理：查看举报内容与来源
  - [ ] 一键处理：删除内容 + 记录用户违规
